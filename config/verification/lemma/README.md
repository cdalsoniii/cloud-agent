# Lemma (optional)

Stub directory for Lemma / LemmaFit source-level specs.

Midspiral business rules live in `rules/business_rules.yaml` and are checked via Mastra `verify-rules`.

When Lemma CLI (`lsc`) is available via `./scripts/setup-verification-tools.sh`, place `*.lemma` files here and they will be picked up by `verify-local.sh`.
